História da Linguagem Dart

Introdução da Linguagem Dart


Dart é uma linguagem de programação desenvolvida pela Google, voltada principalmente para o desenvolvimento web. Ela foi criada pelos engenheiros Lars Bak, conhecido pelo seu trabalho no Google Chrome, e Kasper Lund. A primeira aparição oficial do Dart aconteceu em outubro de 2011, durante a GOTO Conference na Dinamarca. Desde então, a linguagem evoluiu muito e se consolidou como uma ferramenta poderosa e versátil.


A aprovação do Dart pelo comite TC52 da Ecma International


Para que o Dart pudesse ser lançado de forma oficial e reconhecido como uma linguagem de programação padrão, foi necessário passar pela aprovação do comitê técnico TC52 da Ecma International. Esse processo garantiu que o Dart seguisse todos os padrões internacionais de desenvolvimento e fosse aceito por todos os navegadores modernos, o que foi um marco importante para sua popularização.


A atualizaçãopara o Dart 2 em 2018


Em 2018, o Dart passou por uma grande atualização com o lançamento do Dart 2. Essa nova versão trouxe várias melhorias, incluindo uma otimização focada no desenvolvimento Client-Side (do lado do cliente) para a web. Além disso, o Dart também entrou no mundo do desenvolvimento mobile, ganhando ainda mais versatilidade.


As duas soluções atuais: Dart Native e Dart Web, explicando suas funções.


Hoje, o Dart oferece duas grandes soluções: Dart Native e Dart Web. O Dart Native é a escolha certa para quem deseja desenvolver aplicativos mobile, de desktop ou servidores. Ele conta com a máquina virtual Dart e dois tipos de compilação: JIT (just-in-time), que compila o código durante a execução, e AOT (ahead-of-time), que compila o código antes da execução, gerando um código de máquina otimizado.Por outro lado, o Dart Web, como o próprio nome indica, é voltado para o desenvolvimento web. Ele vem com o Dartdevc, um compilador que transforma o código durante o desenvolvimento, e o dart2js, que é usado para compilar o código final que será usado em produção.


A necessidade do Dart SDK para o desenvolvimento, exceto no caso do Flutter.


Para desenvolver utilizando Dart, seja para web, servidores ou scripts de linha de comando, é necessário instalar o Dart SDK. Ele oferece uma biblioteca e várias ferramentas que ajudam no processo de desenvolvimento. Agora, se o foco for o desenvolvimento mobile, não é necessário instalar o SDK do Dart diretamente, basta instalar o Flutter, que é o framework mais popular da linguagem. Um dos grandes diferenciais do Dart é que ele é fortemente tipado, mas oferece a flexibilidade de o desenvolvedor escolher se quer declarar os tipos explicitamente ou deixar que a linguagem faça isso automaticamente através de inferência.


Principais Frameworks


  • Flutter: Muito popular para desenvolvimento mobile e multiplataforma.
  • AngularDart: Mais voltado para desenvolvimento web.
  • Material Design Lite: Para criar interfaces seguindo o padrão do Material Design.
  • OverReact, VueDart, Aqueduct, Angel e Jaguar: São outras opções que ampliam ainda mais as possibilidades de desenvolvimento com Dart, cobrindo áreas como desenvolvimento de APIs, servidores e até interfaces de usuário reativas.
